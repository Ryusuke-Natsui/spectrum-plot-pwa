# Text Plot PWA

2列の数値データを含むテキストファイル（`.txt`, `.csv`, `.dat` など）を読み込み、ブラウザ上で折れ線グラフ表示するシンプルなPWAです。

## できること

- ローカルのテキストファイルを読み込み
- 先頭2列の数値を自動抽出
- 折れ線グラフを表示
- 軸ラベルとタイトルを編集
- グラフをPNG保存
- Service Worker によるオフライン利用
- GitHub Pages にそのまま配置可能

## 想定する入力形式

以下のように、1行につき2つの数値が並ぶ形式を想定しています。

```txt
19764 72.3333
19763.3 71.8889
19762.6 72.1111
```

- 区切りはスペース、タブ、カンマ、セミコロンに対応
- 空行は無視
- `#` / `//` / `;` から始まるコメント行は無視
- 数字以外を含むヘッダー行は自動でスキップ

## GitHub Pages への載せ方

1. GitHub で新しいリポジトリを作る
2. このフォルダの中身をそのままアップロードする
3. GitHub の **Settings → Pages** を開く
4. **Deploy from a branch** を選ぶ
5. Branch を **main**、Folder を **/ (root)** にする
6. 保存すると公開URLが発行される

## カスタマイズ候補

このあと必要なら以下を足せます。

- 複数ファイルの重ね描き
- ズーム・パン
- ピーク検出
- スムージング
- 軸反転
- CSV書き出し
- ラマン用の波数変換や校正機能
